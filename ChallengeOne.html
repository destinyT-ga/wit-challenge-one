Introduction

JavaScript is a cross platform object data oriented scripting language. It is a small and lightweight language. Inside a host environment(for example, a web browser), JavaScript can be connected to the objects of its environment to rpovvide proh=grammatic control over them.

JavaScript contains a standard library of objects such as Array, data, and math as well as set of langauge elements like operattors, control structrues and statements. Core JavaScript can be extended for a variety of purposes by supplementing it with additional objects. Take for example;

  • Client-sdie JavaScript extneds the core langauge by supplying objects to control a browser and its Document Object Model(DOM)
  • Server-side JavaScript exyends core langauge by supllying objects relevant ot running JavaScript on a server.

JavaScript and-Java

JavaScript and Java are similar in some ways but fundamentally different in some ways. The JavaScript langauge resembles Java but doesnt have Java's static typing and strong checking.

In contrast to Java's compilar tome system of classes buikt by Declaring, JavaScript supports a routine system based on a smaller number of data types representing numeric, Boolean and string values.

JavaScript is a very free-form language compared to Java.You dont have to declare all varaianles, classes and methods. You do not have to be concerned with whether methods are public, private or protected and you dont have to implement interfaces, variables, parameters and functions return types are not explicity types.

Variables

You use variables as symbolic names for valuess in your application. The names of the variables, called identifiers, conform to certain rules.

A JavaScript identifier must start with a letter, underscore(-), or dollar sign($); susequent characters can also be digits(0-9). Because JavaScript is case sensitive, letters include characters "A" through "Z"(uppercase) and the characters "a" through "z"(lowercase)

Declaring-variables

You declare variables in three ways;

  • with the keyword var. Eg var x = 42
  • This syntex can be used to declare both local and global variables
  • By simply assigning it a value. eg y=13
This syntex can be used to declare a block scope local variables. See Variable scope below.

Variable-scope

When you declare a variable outside of any fucntion, it is called a global variable, Because it is available to any other code in the current document. When you declare a variable within a function, it's called a local variable, becasue it is available only within thatt function.

JavaScript before ECMAScript 2015 doesnt have block statement scope, rather, a variable delcared within a block is localto the fucntion(or global varaiale scope) that the block resides within if (true){ var x = 5}; //5

This behaviour changes when using the let Declaration in ECMAScript 2015 if(true) {let y= 5;} console.log(y);//ReferenceError:y is not defined.

Global-Variables

Global variables are in fact properties of the global object. In web pages the global object is window, so you can set and acess global variables using window.variable syntax

Consequently, you can access global variables declared in one window or frame from another window or frame by specifying the window

Constants

You can create a read-only, named constant with the constant keyword. The syntax of a constant identifier is the same as foor a variable identifier. It must start with a letter, underscore or dollar sign and can contain alphabetic, numeric or underscore charcaters. constant P1 = 3.14

Data type

The latest ECMAScript standard defines seven data types

Data types that are primitive:

  • Boolean.true and false
  • null. A special keyword denoting a null value
  • Number 42 or 3.14159
  • String "Howdy"
  • Symbolic(new in ECMAScript 2015). A data type whose instances are unique and immutable

Function Declaration

A fucntion definition(also called a function declartion, or function statement) consists of the fucntion keyword, followed by:

  • The name of the fucntion
  • A list of arguments to the function, enclosed in parenthese and separated by commas
  • The JavaScript statement that defines the function, enclosed in curly brackets, {}

For example, the following code defines a simpple function named square: function square(number) {return number*number; }
